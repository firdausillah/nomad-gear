## About

**Nomad Supply Co.** is a fictional travel gear brand created as part of the **Shopify Ecommerce Lab**.

The purpose of this project is **not** to build another demo store.

Instead, it demonstrates how a Software Engineer approaches Shopify development by solving real ecommerce problems through product thinking, clean engineering, and business-oriented decision making.

The project evolves through product sprints, where each release represents a meaningful improvement to the customer experience and business performance.

---

## Project Goals

- Build production-quality Shopify themes.
- Demonstrate real ecommerce problem solving.
- Learn Shopify using real-world scenarios.
- Showcase engineering thinking beyond UI implementation.
- Create portfolio-worthy case studies.
